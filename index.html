Smart TV Keyboard :: Demo Page

Smart TV Keyboard is a simple virtual keyboard for smart TVs. It has no external dependencies. Optimized for using with TV remote control.

Hint: Use arrow keys to navigate. Press Enter to enter selected value. Press Backspace to close window.

Modal Window Example

Hint: Use arrow keys to navigate. Press Enter to enter selected value. Press Backspace to close window.

Languages

You can use your own layouts. Examples of layouts can be found in "lang" directory.
